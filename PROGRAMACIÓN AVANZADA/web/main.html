                                function validarYConvertirHora(hora) {
                                  const regexHora = /^([0-1]?[0-9]|2[0-3]):([0-5][0-9]):([0-5][0-9])$/;
                                  if (regexHora.test(hora)) {
                                    return hora;
                                  } else {
                                    const partes = hora.split(':');
                                    if (partes.length === 3) {
                                      const horas = parseInt(partes[0], 10);
                                      const minutos = parseInt(partes[1], 10);
                                      const segundos = parseInt(partes[2], 10);
                                      if (!isNaN(horas) && !isNaN(minutos) && !isNaN(segundos)) {
                                        const horasValidas = Math.min(Math.max(horas, 0), 23);
                                        const minutosValidos = Math.min(Math.max(minutos, 0), 59);
                                        const segundosValidos = Math.min(Math.max(segundos, 0), 59);
                                        return `${horasValidas.toString().padStart(2, '0')}:${minutosValidos.toString().padStart(2, '0')}:${segundosValidos.toString().padStart(2, '0')}`;
                                      }
                                    }
                                    return '00:00:00';
                                  }
                                }
                                
                                function subirDatosDesdeCSV(rutaCSV) {
                                  const resultados = [];
                                  
                                  fs.createReadStream(rutaCSV)
                                    .pipe(csv())
                                    .on('data', (data) => {

                                        movilizacion: data.movilizacion || 'SIN DATO',
                                        presunta_flagrancia: data.presunta_flagrancia || 'SIN DATO',
                                        presunta_modalidad: data.presunta_modalidad || 'SIN DATO',
                                        flagrante_boleta: 'SIN DATO',
                                        lugar: data.lugar || 'SIN DATO',
                                        tipo_lugar: data.tipo_lugar || 'SIN DATO',
                                        infraccion: data.presunta_infraccion || 'SIN DATO',
                                        subinfraccion: data.presunta_subinfraccion || 'SIN DATO',
                                        anio: new Date(data.fecha_detencion_aprehension).getFullYear(),
                                        Detenido_id: resultados.length + 1,
                                        arma_id_arma: data.nac || 1,
                                        Subcircuito_id_subcircuito: data.codigo_subcircuito || -1

                                            tipo_lugar, infraccion, subinfraccion, anio, Detenido_id, 
                                            arma_id_arma, Subcircuito_id_subcircuito
                                          ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
                                        `;
                                        const valores = [
                                          fila.fecha_detencion, fila.hora_detencion, fila.tipo, fila.condicion,
                                          fila.movilizacion, fila.presunta_flagrancia, fila.presunta_modalidad,

                                        ];
                                        connection.query(query, valores, (err, res) => {
                                          if (err) {
                                            console.error('Error insertando datos:', err);
                                            return;
                                          }
                                        });
                                      });
                                
                                      connection.end();
                                    });

Pregunta 1

Conclusión

En general, como se muestra en la figura, hay una variación significativa en los niveles de pobreza extrema, el número de ocurrencias y el número de infracciones entre las provincias. Existen algunas de las provincias con niveles más altos de pobreza extrema pero un bajo número de ocurrencias, al igual que existe un mayor número de ocurrencias, pero una baja tasa de pobreza extrema o mayor número de infracciones. Por lo tanto, los demás factores que no se representan en la gráfica pueden ser responsables de los indicadores. Existen factores que podrían influir en estos datos, como el acceso a los servicios de salud, la educación y conciencia en los jóvenes y factores externos socioeconómicos.

Pregunta 2

Conclusión

En la figura, se puede observar que algunas provincias mayores tasas de incautación de droga conllevan un gran número de población colombiana, y otras poblaciones pequeñas, pueden tener una elevación de tasa de incautación de drogas. Investigando, sabemos que la población colombiana tiene un alto expediente de exportación de droga a diferentes países vecinos, siendo esta más notoria en la época de la pandemia. Es normal que se pueda relacionar a la nacionalidad colombiana con esta tasa de narcotráfico, esto podría influir en la percepción de las autoridades y su enfoque en diferentes provincias hacia la incautación de drogas en provincias donde haya una mayor población colombiana.

Pregunta 3

Conclusión

se puede observar una diversidad significativa en cada provincia. Estas diferencias pueden deberse a varios factores, uno de ellos puede ser la eficacia que tienen las autoridades o fuerzas de seguridad en las distintas provincias. Esto nos indica y los logra diferenciar qué etnias predominan en cada provincia, tomando en cuenta su cultura y la cantidad de población que contiene cada una.

Pregunta 5

Conclusión

En la figura, se muestran varias provincias, algunas de ellas con una mayor tasa de empleo inadecuado, mientras que otras presentan cifras bajas al igual que un menor número de detenciones. El empleo inadecuado nos presenta un reflejo del porcentaje de la población que trabajan en empleos que no cumples con sus necesidades económicas, estos datos varían porque algunas provincias pueden enfrentar un desafío económico, puede encontrarse una falta de diversidad en el ámbito laboral y una gran demanda de trabajadores en diferentes áreas de trabajo. La calidad de educación también llega a ser un factor importante, ya que, si las personas no cumplen con las habilidades necesarias, acaban en empleos que no favorecen al trabajador y puede conllevar a una mala paga. Por ejemplo, en la provincia de Manabí el nivel de instrucción más común llega a ser la educación inicial, pero presentando una alta tasa de empleo inadecuado al igual que un gran número de detenciones. Se puede llegar a la conclusión de que, al no ser más común un nivel de instrucción alto el, la falta de educación desde la niñez puede afectar significativamente a la carencia de habilidades que pueden desarrollar las personas para conseguir un trabajo adecuado. Puede también, con llevar a realizar otro tipo de actos vandálicos e ilícitos que con lleven a su detención. En resumen, las detenciones pueden relacionarse directamente con la economía y la educación. Estos datos pueden ayudar a diseñar políticas o planes para abordar estos problemas en cada provincia.
